## JSON Literals

- string: `"some string"`
- number: `2`, `1.34`, ...
- boolean: `true`, `false`
- null: `null`
- array: `[]`
- object: a `JSON` in your `JSON`
    > In Doc we should define our new types that are our object types. \
    > Value types are shown in comments. (if no comment value is string as default)
