# PolyLineData types

* Id
* Name
* Color
* Width
* Command
* LayerId
* Points

# Example1

```JSON
{
  "Id": 100,
  "Name": "some name",
  "Color": "red",
  "Width": 1000,
  "Command": "Update",
  "LayerId": 120,
  "Points":[
      "Longitude": 2,
      "Latitude": 2,
      "Altitude": 2
  ]
}
```
